| T-04-07 | T | Path traversal in ingest | mitigate | ingest only reads operator-supplied paths; backend validates uploads separately |
| T-04-08 | I | API_KEY in CLI stdout | mitigate | Never print API_KEY; Bearer only in request headers |
| T-04-09 | D | Unbounded directory walk | accept | Operator-supplied path; skip non-allowlisted extensions |
