<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HTTP client → /api/v1/* | Untrusted requests; optional Bearer token when AUTH_ENABLED |
| Config load → process.env | API_KEY secret must not log or expose in errors beyond field name |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | S | Missing auth on admin API | mitigate | preHandler verifyBearerAuth on all /api/v1/* when AUTH_ENABLED (CONF-03) |
| T-04-02 | I | Timing attack on API key compare | mitigate | Use @fastify/bearer-auth constant-time compare — do not hand-roll |
| T-04-03 | I | API_KEY logged on startup | mitigate | Never log config.API_KEY; SECRET_KEYS already redacts in formatConfigError |
| T-04-04 | E | Global bearer hook breaks health | mitigate | addHook: false; health/swagger registered without preHandler |
| T-04-05 | T | Unauthorized upload/delete/search | mitigate | routeOpts on documents.ts and search.ts all methods |
</threat_model>
