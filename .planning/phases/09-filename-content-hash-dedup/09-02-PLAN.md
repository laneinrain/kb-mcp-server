  <action>
    Extend **`IngestResult`** with `outcome: 'created' | 'unchanged' | 'replaced'`.

    Refactor **`ingest()`**:
    1. Parse file → `text`, compute `contentHash`
    2. Resolve `filename` (options.filename ?? parsed)
    3. `existing = registry.findByUserAndFilename(userId, filename)`
    4. If existing && existing.contentHash === contentHash → return `{ outcome: 'unchanged', documentId: existing.id, chunkCount: existing.chunkCount, collection: existing.collection }`
    5. `documentId = existing?.id ?? deriveDocumentIdForUserFile(userId, filename)`
    6. If existing && hash differs → deleteByDocumentId, set outcome `replaced`; else `created`
    7. registerDocument / chunk / embed / upsert (unchanged path skips steps 6–7 embed)

    Remove use of path-based **`deriveDocumentId(absolutePath)`** from ingest flow (keep path guard only).

    Update **`sourcePath`** on row to latest upload path.
  </action>
